To use PHP project you have to first import database into your own system through XAMPP, for that follow the instruction.
1.	Open XAMPP Control Panel
2.	Hit the start buttons from Apache and MySQL Module column
3.	Open browser and go to localhost/phpmyadmin
4.	Create a new database at your own (make sure database will be having the same name of .sql file)
5.	Find the import section in the header of that created database
6.	Choose database(.sql) file
7.	Click Go.
8.	Now go to 
		localhost/Movie-Ticket-Booking/Site -> for user
		localhost/Movie-Ticket-Booking/Dashboard/admin-login.php -> for admin
